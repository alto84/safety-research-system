# Literature Review Audit Checklist

## Overview
This checklist is used by the Literature Auditor to validate literature review outputs.

## Anti-Fabrication Checks (CRITICAL)

### Score Fabrication
- [ ] No numerical scores above 80% without external validation
- [ ] No composite scores without calculation basis
- [ ] No letter grades (A, B, C) without defined rubric
- [ ] All scores have documented measurement methodology

### Banned Language
- [ ] No "exceptional performance" without extraordinary evidence
- [ ] No "outstanding" or "world-class" claims
- [ ] No "industry-leading" without baseline measurements
- [ ] No "X times better" without documented baseline

### Evidence Requirements
- [ ] Primary sources only (not AI-generated citations)
- [ ] Measurement data provided for all numerical claims
- [ ] No citation of other AI outputs as evidence
- [ ] Statistical rigor documented where applicable

## Completeness Checks

### Required Fields
- [ ] Summary present and substantive (>50 characters)
- [ ] Sources list provided with at least 2 sources
- [ ] Evidence level documented
- [ ] Confidence level explicitly stated
- [ ] Limitations section present
- [ ] Methodology described

### Source Quality
- [ ] Each source has title
- [ ] Each source has authors
- [ ] Each source has publication year
- [ ] Each source has identifier (PMID, DOI, or URL)
- [ ] Minimum 2 sources reviewed (warning if fewer)

## Evidence Grading

### Evidence Level Validation
- [ ] Evidence level matches source types
- [ ] Hierarchy properly applied:
  - Level 1: Systematic reviews, meta-analyses, RCTs
  - Level 2: Cohort studies
  - Level 3: Case-control studies
  - Level 4: Case series, expert opinion
- [ ] Confidence level aligns with evidence level
- [ ] No high confidence with low evidence level

## Uncertainty Expression (REQUIRED)

### Confidence Level
- [ ] Confidence explicitly stated (low/moderate/high)
- [ ] Confidence justified by evidence
- [ ] Uncertainty acknowledged
- [ ] Limitations documented

### Limitations
- [ ] At least 2 limitations identified
- [ ] Limitations are specific (not generic)
- [ ] Missing data acknowledged
- [ ] Biases noted where applicable

## Search Methodology

### Methodology Documentation
- [ ] Databases searched listed
- [ ] Search terms documented
- [ ] Inclusion criteria defined
- [ ] Exclusion criteria defined
- [ ] Date range specified

### Transparency
- [ ] Total results reported
- [ ] Included/excluded counts provided
- [ ] Exclusion reasons documented
- [ ] Search reproducible from documentation

## Citation Quality

### Citation Completeness
- [ ] All sources have unique identifiers
- [ ] Publication years within reasonable range
- [ ] Authors listed (not "Anonymous")
- [ ] Titles descriptive and specific

### Source Credibility
- [ ] Sources are peer-reviewed where claimed
- [ ] Pre-print status disclosed if applicable
- [ ] Conflicts of interest noted if relevant
- [ ] Industry-sponsored studies labeled

## Synthesis Quality

### Evidence Synthesis
- [ ] Findings accurately represent sources
- [ ] Conflicting evidence acknowledged
- [ ] Gaps in literature identified
- [ ] Strength of evidence characterized

### Bias Assessment
- [ ] Publication bias considered
- [ ] Selection bias acknowledged
- [ ] Confirmation bias avoided
- [ ] Multiple perspectives represented

## Regulatory Alignment

### Safety Standards
- [ ] Appropriate for regulatory submission (if applicable)
- [ ] Evidence hierarchy follows FDA/EMA guidelines
- [ ] Causality assessment framework used (if applicable)
- [ ] Pharmacovigilance standards met (if applicable)

## Issue Severity Classification

### Critical Issues (Must Fix)
- Score fabrication
- Banned language without evidence
- Missing evidence chain
- High confidence with low evidence
- Methodology not documented

### Warning Issues (Should Fix)
- Incomplete sources (< 2)
- Missing confidence level
- Insufficient limitations
- Missing source identifiers
- Evidence level mismatch

### Info Issues (Nice to Have)
- Additional sources would strengthen
- More detailed methodology recommended
- Consider additional databases
- Expand limitations discussion

## Recommended Actions by Issue Type

### If Score Fabrication Found
1. Remove fabricated scores
2. Document actual measurement basis
3. Use evidence-based language only
4. Mark as CRITICAL - retry required

### If Evidence Insufficient
1. Expand literature search
2. Include additional databases
3. Extend date range if appropriate
4. Document gaps explicitly

### If Confidence Overstated
1. Lower confidence level
2. Add qualifying language
3. Expand limitations section
4. Document uncertainty

### If Methodology Unclear
1. Document search strategy
2. List all databases and terms
3. Provide inclusion/exclusion criteria
4. Make search reproducible

## CLAUDE.md Compliance

All literature reviews must comply with CLAUDE.md:
- Truth over positivity
- Evidence over opinion
- Uncertainty over false precision
- Skepticism over optimism
- No fabrication, ever
