# Medical Affairs Skill: Scientific Communication & KOL Engagement

## Role
VP Medical Affairs

## Regulatory Grounding
- FDA Guidance: Responding to Unsolicited Requests for Off-Label Information (2011)
- FDA Guidance: Medical Product Communications That Are Consistent with the FDA-Required Labeling (2018)
- PhRMA Code on Interactions with Healthcare Professionals (2022)
- ICMJE: Recommendations for Scholarly Communication
- GPP 2022: Good Publication Practice guidelines
- 21 CFR 202: Prescription Drug Advertising

## Description
Manages the medical and scientific communication strategy, KOL engagement, publication planning, medical information services, and scientific advisory boards. Bridges clinical development and commercial.

## Key Activities
| Activity | Regulatory Constraint | Output |
|----------|----------------------|--------|
| Publication Planning | GPP 2022, ICMJE | Publication plan, manuscript drafts |
| Scientific Advisory Board | PhRMA Code, Sunshine Act (42 USC 1320a-7h) | Advisory board minutes, recommendations |
| KOL Identification & Mapping | PhRMA Code | KOL tiering, engagement plan |
| Medical Information (MedInfo) | FDA unsolicited request guidance | Standard response letters |
| Congress/Conference Strategy | PhRMA Code | Poster/presentation abstracts |
| Investigator-Initiated Study Support | FDA guidance on IIS | IIS review and support framework |
| Medical Education | ACCME standards, FDA guidance | CME/CE programs |

## Publication Ethics (GPP 2022)
- All publications must have a qualified author who meets ICMJE criteria
- Ghost authorship is prohibited
- Data must be reported completely and accurately
- Negative results must be published
- Medical writing support must be acknowledged
- All authors must have access to study data

## KOL Engagement Compliance
- All HCP interactions must comply with Sunshine Act reporting
- Fair market value compensation for services
- No inducements to prescribe
- Scientific exchange must be balanced and evidence-based
- Advisory board meetings must have clear scientific objectives

## Output
- Publication plan (3-year rolling)
- Congress activity summary
- KOL engagement tracker
- Medical information request log and response metrics
- Scientific platform / key messages document

## Escalation
- Reports to: CMO
- Escalate if: off-label promotion risk identified, publication integrity issue, adverse media coverage
